
# Life Expectancy Prediction and Clustering using Machine Learning

## Overview

This project explores the use of supervised and unsupervised machine learning techniques to analyze global life expectancy data. The objective is to understand how different socioeconomic and health-related factors influence life expectancy and to identify patterns among countries.

The project applies regression models to predict life expectancy and clustering algorithms to identify groups of countries with similar characteristics.

---

## Objectives

The main goals of this project are:

* Predict life expectancy using machine learning regression models
* Analyze the relationship between economic and health indicators
* Identify clusters of countries based on similar life expectancy patterns
* Compare different machine learning models and evaluate their performance

---

## Dataset

The dataset contains information about life expectancy across multiple countries, including variables such as:

* GDP
* Adult Mortality
* Infant Deaths
* Health Indicators
* Economic Indicators

These features help explain differences in life expectancy across countries.

---

## Methodology

### Supervised Learning (Regression)

Several regression models were implemented to predict life expectancy using different features.

Models explored include:

* Simple Linear Regression
* Polynomial Regression
* Multiple Linear Regression
* Other supervised learning models for comparison

Polynomial regression was used to capture **non-linear relationships**, particularly between GDP and life expectancy.

---

### Unsupervised Learning (Clustering)

Clustering algorithms were used to identify groups of countries with similar life expectancy characteristics.

Algorithms implemented:

* K-Means Clustering
* Hierarchical (Agglomerative) Clustering
* DBSCAN

These methods help reveal patterns in global health and economic conditions.

---

## Model Evaluation

Different evaluation techniques were used to compare models, including:

* R² score
* Mean Squared Error
* Cluster evaluation metrics
* Visual analysis using scatter plots and clustering visualizations

These methods help determine which models best explain the life expectancy patterns.

---

## Key Insights

The analysis revealed several important insights:

* Economic indicators such as **GDP** strongly influence life expectancy.
* Health indicators such as **adult mortality** and **infant deaths** are significant predictors.
* Some relationships between variables and life expectancy are **non-linear**.
* Clustering techniques can group countries into meaningful health and development categories.
